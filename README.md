# Sales-Data-Dashboard# 📊 Sales Dashboard (2020–2024)

1. 🚀 Project Title
Interactive Sales Analytics Dashboard (2020–2024)

---

2. 📝 Short Description
This project is an interactive Sales Dashboard built to analyze and visualize sales performance from 2020 to 2024. The dashboard provides clear insights into revenue, cost, profit, quantity, order status, brand performance, and geographic sales distribution across India.

It helps stakeholders monitor key performance indicators (KPIs), identify trends, and make data-driven business decisions.

---

3. 🛠 Tech Stack
- Power BI (Dashboard Development & Visualization)
- Microsoft Excel / CSV (Data Cleaning & Preparation)
- DAX (Data Analysis Expressions for KPIs & Measures)
- Data Modeling (Relationships & Schema Design)

---

4. 📂 Data Source
Structured sales dataset (2020–2024) containing:
- Order Details
- Sales Amount
- Cost
- Profit
- Order Status (Delivered, Processing, Shipped)
- Brand Information
- Geographic Location Data

*Note: Dataset used for learning and analytical purposes.*

---

5. 📌 Features
- 📈 Year-wise Sales Analysis (2020–2024)
- 📊 KPI Cards:
  - Total Sales
  - Total Cost
  - Total Profit
  - Total Quantity
  - Order Count
- 📦 Order Status Breakdown (Delivered / Processing / Shipped)
- 🏷 Brand-wise Sales Distribution (Pie Chart)
- 🗺 Geographic Sales Visualization (India Map)
- 👤 Supervisor-based Filtering
- 🎛 Interactive Slicers & Dynamic Filtering

---

6. ✨ Highlights
- Clean and professional dashboard UI
- Fully interactive visuals
- Business-focused KPI tracking
- Geo-spatial visualization for regional analysis
- Insight-driven storytelling through data
- Optimized data model for better performance

---

7. 📊 Key Insights Generated
- Identified top-performing brands by revenue
- Analyzed regional sales concentration
- Tracked order fulfillment distribution
- Compared sales, cost, and profit trends
- Measured supervisor-level contribution

---

8. 🎯 Business Use Case
This dashboard can be used by:

It helps in monitoring performance, improving strategy, and increasing profitability.
